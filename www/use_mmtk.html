Getting Started: VMKit and MMTk

This page gives you the instructions on how to build VMKit with a MMTk garbage collector.

  1. Checkout and build LLVM-GCC from SVN head.
  2. Update PATH to include the llvm-gcc binary.
  3. Re-configure LLVM.
  4. Configure VMKit with three additional options: --with-gc=mmtk, --with-llvmgcc, and --with-mmtk-plan={marksweep|copyms}
  5. Build VMKit:
  6. make ENABLE_OPTIMIZED=1
  7. Run VMKit:
